**A personalized, dynamic portfolio page designed with a distinctive cyberpunk/hacker aesthetic.**

This project showcases academic background, professional experience, and technical skills through a visually striking, retro-terminal interface. The custom CSS and JavaScript create an immersive environment that is both modern and nostalgic.

---

## ‚ú® Features

* **Matrix Rain Background:** A full-screen HTML Canvas driven by JavaScript, displaying a classic "Matrix" digital rain effect.
* **Terminal Aesthetic:** Utilizes custom fonts (**Bitcount** and **Anonymous Pro**) and a dark theme to fully immerse the user in a command-line interface experience.
* **Blinking Cursor:** Features a custom CSS animation to simulate a realistic, command-line blinking cursor.
* **High-Contrast Design:** A color palette built on black, white, and a vibrant accent green (`#00cc33`) for high readability in a dark environment.
* **Detailed Experience:** Dedicated sections covering academic achievements and professional internships (e.g., SEBN-MA), presented in a clean, terminal-like format.
